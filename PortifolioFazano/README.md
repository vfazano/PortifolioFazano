🌐 Portfólio Web | Vitor Fazano

Meu site pessoal desenvolvido para apresentar meus projetos, habilidades e trajetória como desenvolvedor.

📌 Sobre o Projeto

Este é meu portfólio pessoal, criado com o objetivo de mostrar meus conhecimentos em desenvolvimento web e compartilhar meus projetos.

O site apresenta:

👨‍💻 Informações sobre mim

📂 Projetos desenvolvidos

🛠️ Habilidades técnicas

📞 Contato profissional

Além disso, o projeto foi feito com foco em um design moderno, simples e funcional.

🚀 Funcionalidades

✔️ Navegação por seções (scroll suave)

✔️ Apresentação de projetos com links para o GitHub

✔️ Layout organizado e intuitivo

✔️ Animações simples com CSS

✔️ Interface limpa e moderna

🛠️ Tecnologias Utilizadas

💻 HTML5
